Automatically installs 120+ standard amenities for rental properties.

== Description ==

This plugin automatically adds all standard amenities to your Rently property system when activated.

Includes 120+ amenities across categories:
* Internet & Technology (8 amenities)
* Kitchen & Dining (12 amenities)
* Bathroom (8 amenities)
* Bedroom & Laundry (9 amenities)
* Climate Control (4 amenities)
* Outdoor & Views (12 amenities)
* Parking & Transportation (6 amenities)
* Safety & Security (7 amenities)
* Family-Friendly (8 amenities)
* Accessibility (5 amenities)
* Pool & Recreation (8 amenities)
* Pet-Friendly (5 amenities)
* Additional Services (6 amenities)
* Unique Amenities (9 amenities)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ordivo-rently-amenities-installer/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. All amenities will be automatically installed
4. You can deactivate the plugin after installation (amenities will remain)

== Usage ==

1. Activate the plugin
2. Check the admin notice for installation confirmation
3. Go to Properties > Amenities to view all installed amenities
4. Deactivate the plugin (optional - amenities will stay)

== Frequently Asked Questions ==

= Can I deactivate the plugin after installation? =
Yes! Once activated, all amenities are permanently added to your database. You can safely deactivate or delete the plugin.

= What if I already have some amenities? =
The plugin checks for existing amenities and only adds new ones. Existing amenities won't be duplicated.

= Can I delete amenities I don't need? =
Yes, go to Properties > Amenities and delete any amenities you don't want to use.

= Can I add more amenities later? =
Absolutely! Go to Properties > Amenities and add custom amenities anytime.
