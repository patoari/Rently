Real-time messaging widget for host-guest communication.

== Description ==

Ordivo Rently Messaging provides a real-time chat widget for communication between hosts and guests.

Features:
* Real-time chat functionality
* Host ↔ Guest messaging
* Message history
* Chat popup modal
* Mobile full-screen support
* Shortcode: [rently_chat]

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ordivo-rently-messaging/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode [rently_chat] on any page or post

== Usage ==

Add the shortcode [rently_chat] to any page or post where you want the messaging widget to appear.

The widget will display as a floating chat button in the bottom-right corner.
