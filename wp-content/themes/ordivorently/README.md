# Ordivorently Theme

A modern, Airbnb-inspired WordPress theme for property rentals.

## Features

- Clean, minimal design inspired by contemporary booking platforms
- Fully responsive and mobile-optimized
- Property search with advanced filters
- Location hierarchy (Division → District → Thana → City)
- Property categories (Apartment, Villa, Resort, etc.)
- Booking system integration
- User-friendly property submission
- SEO optimized
- Fast loading performance

## Installation

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through WordPress admin
3. Install required plugins:
   - Rently Booking System
   - Rently Property Submission
4. Go to Settings → Permalinks and save to flush rewrite rules

## Required Plugins

- Rently Booking System
- Rently Property Submission

## Theme Support

- Custom Logo
- Custom Menus
- Featured Images
- Widgets
- Custom Post Types (Properties)
- Custom Taxonomies (Property Categories)

## Customization

### Colors
Edit the CSS variables in `style.css`:
- `--primary`: Main brand color
- `--secondary`: Secondary accent color
- `--dark`: Text color
- `--gray`: Muted text color

### Menus
- Primary Menu: Main navigation
- Footer Menu: Footer links

### Widgets
- Property Sidebar: Shown on property pages
- Footer Widget Areas: 4 columns
